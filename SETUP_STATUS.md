## ⚠️ Important Notes

### Hono API Integration
The Hono API routes are set up in `src/app/api/[[...route]]/route.ts`. However, **TanStack Start may need additional configuration** to properly serve these routes. You may need to:

1. **Option A**: Use TanStack Start's server functions instead (not recommended per your preference)
2. **Option B**: Configure Vite dev server to proxy `/api/*` requests to Hono
3. **Option C**: Mount Hono app in a TanStack Start server entry point

### Route Tree Generation
The route tree (`src/routeTree.gen.ts`) is auto-generated by TanStack Router. It will be regenerated when you run `bun run dev`.
